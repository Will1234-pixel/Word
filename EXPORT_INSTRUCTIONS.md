# Exporting or Downloading `ControlCt_Routine_FBD.L5X`

You can retrieve the routine file directly from the repository using any of the methods below.

## 1) Download a single file from GitHub (browser)
1. Open the repository in GitHub.
2. Navigate to `ControlCt_Routine_FBD.L5X`.
3. Click **Download raw file** (or press **Raw** and save the page) to save the L5X locally.

## 2) Download via command line with `curl`
Run the command below, replacing `<BRANCH>` with the branch name if different:
```bash
curl -L -o ControlCt_Routine_FBD.L5X "https://raw.githubusercontent.com/<OWNER>/<REPO>/<BRANCH>/ControlCt_Routine_FBD.L5X"
```

## 3) Export from a local clone
If you already have the repository cloned locally, you can copy the file out:
```bash
# From inside the repo root
cp ControlCt_Routine_FBD.L5X /path/where/you/want/ControlCt_Routine_FBD.L5X
```

## 4) Create an export from a specific commit
To export the file as it existed at a particular commit or tag:
```bash
# Replace <REF> with the commit SHA or tag
git show <REF>:ControlCt_Routine_FBD.L5X > ControlCt_Routine_FBD.L5X
```

## 5) Download the entire repository
If you prefer to download everything as a ZIP:
1. Open the repository in GitHub.
2. Click **Code** > **Download ZIP**.
3. Extract the archive and locate `ControlCt_Routine_FBD.L5X` inside.

These options let you grab the L5X file whether you are browsing on the web or working from the command line.
